# Forza Horizon 6 Local Dataset

This archive contains:

- `cars.json`: structured car data, FH6 game stats, specs, sources, and image metadata
- `cars.csv`: flat table for importing into a database/spreadsheet
- `images/`: downloaded main images referenced by the wiki pages
- `local_game_packages.json`: installed local game car package list from `media/Cars/*.zip`

Important: the image and game data sources are tracked inside `cars.json`. Review licensing before publishing a public or commercial site.
